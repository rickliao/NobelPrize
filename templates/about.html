Data

Nobel Prize API

We used requests from python to get information from the API. For example, a get call to http://api.nobelprize.org/v1/prize.json?year=2008&category=economics will return the information in json for nobel prize in economics in 2008. For each prize, the API also return the laureate's id, which is used to get laureate information from the same API. A get call like http://api.nobelprize.org/v1/laureate.json?id=919 returns the information associated with the laureate whose id is 919.

Countries API

We used this API to get information on coutnries. Each laureate returned from Nobel Prize API contains a country code, and we use this country code to scrap data of the country. For example, a get call to https://restcountries.eu/rest/v1/alpha/us will return the information in json on country with country code us, which is United States.
